# Zauri Roadmap

*Last updated: 2026-03-28*

## Next Up

- [ ] **Multi-cursor editing** — Cmd+D select next occurrence, Cmd+Shift+L select all, column selection
- [ ] **Command palette** — Cmd+Shift+P fuzzy search to run any action (open file, switch branch, settings, git, panels)
- [ ] **Tab drag reorder + split view** — drag tabs to reorder, side-by-side editing of two files
- [ ] **Minimap** — scrollbar code preview showing full file structure
- [ ] **Breadcrumbs** — `src > components > Editor.tsx > render()` path bar using LSP/tree-sitter document symbols
- [ ] **Extensions/plugins** — user-installable plugins without forking
- [ ] **Themes** — light mode, custom color schemes, VS Code theme import
- [ ] **Image/PDF preview** — preview non-code files inline
- [ ] **Collaborative editing** — real-time multiplayer via CRDTs
- [ ] **Worktree support** — isolated git worktrees per AI thread (t3code-style)
